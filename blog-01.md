# ตอนที่ 1: จุดเริ่มต้นของฉัน

สวัสดีครับ! นี่เป็นบทความแรกในบล็อกของผมที่สร้างด้วย GitHub Pages 

## ทำไมผมถึงสร้างบล็อกนี้?
1. เพื่อบันทึกการเรียนรู้ด้านโปรแกรมมิ่ง
2. เพื่อฝึกใช้งาน GitHub ให้คล่องแคล่ว
3. เพื่อแชร์ประสบการณ์ให้กับเพื่อนๆ

### สิ่งที่ผมได้เรียนรู้ในวันนี้
* การสร้าง Repository
* การตั้งค่า GitHub Pages
* การใช้งาน Markdown เบื้องต้น
